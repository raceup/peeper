Install
-------

Different ways, all equals

via ``pipenv``
~~~~~~~~~~~~~~

-  ``$ pipenv install .``
-  ``$ make install``

via ``pip``
~~~~~~~~~~~

-  ``$ pip3 install raceup-peeper``
-  ``$ make pip-install``

fast install
~~~~~~~~~~~~

-  ``make fast-init`` *just copies source files to distribution files
   … run it only if you’re sure about dependencies*

.. _pip: https://pypi.org/project/raceup-peeper/
